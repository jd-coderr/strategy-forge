# Strategy Forge

AI-Powered Trading System Generator

## Overview

Strategy Forge is an AI-powered trading strategy optimization platform that generates, tests, validates, and ranks trading systems across multiple timeframes and risk profiles.

The platform combines:

- Live market intelligence
- Strategy generation
- Backtesting
- Risk analytics
- Performance validation
- Strategy optimization

to identify the highest quality trading systems based on return, drawdown, expectancy, and risk-adjusted performance.

## Features

- Live CoinMarketCap market intelligence
- Automated strategy generation
- Multi-strategy optimizer
- Equity curve visualization
- Risk analytics
- Trade history reporting
- Performance benchmarking against buy-and-hold
- Competition compliance checks
- Wallet connectivity

## Technology Stack

Frontend:
- React
- Vite
- Recharts

Backend:
- FastAPI
- Python

Data Sources:
- CoinMarketCap API

## How It Works

1. Select asset
2. Select timeframe
3. Select risk profile
4. Generate strategy or run optimizer
5. Review backtest results
6. Analyze risk metrics
7. Deploy selected strategy

## Risk Metrics

- Sharpe Ratio
- Sortino Ratio
- Calmar Ratio
- Profit Factor
- Expectancy
- Maximum Drawdown
- Recovery Factor

## Deployment

Frontend:
- Vercel

Backend:
- Railway
